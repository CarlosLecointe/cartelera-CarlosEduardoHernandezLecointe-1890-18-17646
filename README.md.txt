# 🎬 Cartelera de Cine – Frontend

Proyecto frontend desarrollado como práctica académica para consumir la **API pública de cartelera de cine**, mostrando búsqueda, filtros, detalle de películas y un panel de administración (CRUD).

---

## 🚀 Tecnologías utilizadas
- **Vue 3 + Vue Router** → Componentes reactiv@s y navegación SPA.  
- **Bootstrap 5** → Estilos modernos, responsivos y adaptables a móviles.  
- **JavaScript (ES6+)** → Manejo de estado y lógica de consumo de API.  
- **HTML5 + CSS3** → Estructura semántica y personalización visual.  
- **LocalStorage** → Persistencia local en caso de que la API bloquee ciertas operaciones.

---

## 📌 Funcionalidades
- **Búsqueda** por título y ubicación.  
- **Visualización** dinámica de cartelera (tarjetas de películas).  
- **Detalle** completo de una película (`imdbID`).  
- **CRUD (Admin)**: agregar, editar y eliminar películas (intento contra API; fallback local si la API bloquea).  
- **Diseño responsivo** con modo claro/oscuro estilizado.
